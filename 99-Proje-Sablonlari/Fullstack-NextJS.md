# 🌐 Full-stack Next.js & Supabase Şablonu

## 📁 Klasör Yapısı
`	ext
/
├── app/                # Next.js App Router (Sayfalar, Layoutlar)
├── components/         # UI Bileşenleri (Shadcn/UI, Shared)
├── lib/                # Yardımcı Fonksiyonlar (Utils, Config)
├── hooks/              # Custom React Hooks
├── services/           # API ve Veritabanı Çağrıları (Supabase Clients)
├── types/              # TypeScript Tanımlamaları
├── public/             # Statik Varlıklar (İkonlar, Resimler)
└── supabase/           # Migrations ve Veritabanı Şemaları
`

## 🛠️ Temel Araç Seti
- **Frontend:** Next.js, Tailwind CSS, Shadcn/UI
- **Backend:** Supabase (Auth, DB, Storage)
- **State:** TanStack Query / Zustand
- **Validation:** Zod
