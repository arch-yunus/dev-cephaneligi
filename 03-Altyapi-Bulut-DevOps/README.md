# 🛡️ Altyapı, Bulut & DevOps

Yazılımın güvenli, ölçeklenebilir ve sürekli çalışmasını sağlayan operasyonel katman.

## 💡 Neden Buradayız? (Merak Uyandır!)
"Benim bilgisayarımda çalışıyordu" bahanesini tarihe gömen disiplindeyiz. Milyonlarca kullanıcıya hizmet veren bir sistemin nasıl tek bir saniye bile aksamadan ayakta kaldığını merak ettiniz mi? DevOps, kodun yazıldığı andan kullanıcıya ulaştığı ana kadar olan o devasa orkestrasyonun şefidir.

## 🛠️ Nasıl Başlamalıyım? (Yönlendir!)
1. **Linux & Terminal:** Komut satırında ustalaşın.
2. **Git & GitHub Actions:** CI/CD süreçlerini anlamak için otomatik testler kurun.
3. **Docker:** Uygulamaları konteynerize etmeyi öğrenin.
4. **Bulut Temelleri:** Bir AWS veya DigitalOcean VPS üzerinde basit bir web sunucusu yayına alın.
